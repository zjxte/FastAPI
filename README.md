# FastAPI

## set Virtual environment
    
    python -m venv venv
    venv\Scripts\activate.bat
    
    venv\Scripts\activate.ps1

## Install dependencies

    pip install fastapi[all]
    pip install uvicorn
    pip install pydantic
    
## check installed packages

    pip freenze

## run the server
    python -m uvicorn main:app --reload
